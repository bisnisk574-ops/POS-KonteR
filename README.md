Berikut adalah deskripsi singkat untuk aplikasi **KONTERKU**:

KONTERKU adalah aplikasi manajemen bisnis pulsa dan paket data berbasis Google Apps Script yang terintegrasi langsung dengan Google Sheets. Aplikasi ini memudahkan pencatatan transaksi harian, pemantauan stok, serta pembuatan laporan keuangan secara otomatis dan *real-time*. Dengan KONTERKU, pengelolaan operasional konter menjadi lebih praktis, rapi, dan dapat diakses dengan mudah dari mana saja.
